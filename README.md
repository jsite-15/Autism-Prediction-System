# 🧠 Autism Prediction System using Machine Learning

## 📌 Overview

The Autism Prediction System is a Machine Learning application that predicts the likelihood of Autism Spectrum Disorder (ASD) based on screening questionnaire responses and demographic information.

The project covers the complete machine learning workflow, from data preprocessing to model deployment using Streamlit.

---

## 🚀 Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Label Encoding
* SMOTE for handling class imbalance
* Decision Tree Classifier
* Random Forest Classifier
* XGBoost Classifier
* Hyperparameter Tuning using RandomizedSearchCV
* Cross Validation
* Feature Importance Analysis
* Interactive Streamlit Web Application
* Predictive System with Confidence Score
